# Habit Tracker (Widget)

A simple habit tracker that helps anyone track daily/weekly habits, maintain streaks, and view a 7-day history.

## Features
- Add daily/weekly habits
- Mark habits complete for today
- Streak counter (consecutive active days)
- 7-day history view
- Export/Import data (backup & move between devices)
- Works offline (LocalStorage)

## Run locally
Just open `index.html` in a browser.
